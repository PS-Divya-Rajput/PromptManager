# PromptVault

PromptVault is a command-line application developed using Python and Object-Oriented Programming (OOP). It helps users store, organize, search, edit, and manage AI prompts.

## Features

- Add Prompt
- View Prompts
- Search Prompt
- Edit Prompt
- Delete Prompt
- Toggle Favorite
- Save Data in JSON
- View Statistics

## OOP Concepts Used

- Functions
- Classes and Objects
- Encapsulation
- Inheritance
- Polymorphism

## Technologies

- Python 3
- JSON
- VS Code
